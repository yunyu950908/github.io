## 前后端开发联调注意事项：

- 约定数据：有哪些需要传输的数据，数据类型是什么；
- 约定接口：确定接口名称及请求和响应的格式，请求的参数名称、响应的
- 数据格式：根据这些约定整理成接口文档

## 后端接口完成前如何自行mock数据：

- 安装 nodejs => npm 安装 server-mock（当然你也可以用 easy-mock 等其他工具，对新手都比较友好）
- 按约定的数据类型，编写路由文件（router.js）及模板文件（user.ejs）
- 在本地使用 server-mock 来模拟数据。
